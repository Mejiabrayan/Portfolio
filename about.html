About Me

I started my journey as a self-taught web developer about 4 years ago. I had no prior experience with coding or web development, and I definitely didn’t have a degree in computer science. Nevertheless, I was determined to teach myself how to code and build websites. It took a lot of hard work, but I eventually became proficient in HTML, CSS, and JavaScript. I also learned how to use various programming frameworks and libraries. And most importantly, I learned how to think like a programmer. These days, I'm currently a full-time student at CareerFoundry and plan on graduating this Decemeber of 2022, and aim to land my first full-time role as a full-time web developer and I couldn’t be happier. I want to inspire anyone who is thinking of becoming a Web Developer and breaking into tech that nothing is impossible if you set your mind to it. If you’re passionate about coding and web development, then you too can be a self-taught web developer!
